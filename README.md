# calculadora-java
Calculadora "manual" feita em Java com todas funções básicas.

Utilização:

1 = Soma
2 = Subtração
3 = Divisão
4 = Multiplicação
5 = Exponenciação
6 = Radiciação

Após operação:

0 = Encerrar
1 = Continuar

Observações:

A radiciação só opera raizes quadradas e com precisão de 0.01.
